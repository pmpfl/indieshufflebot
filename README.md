# IndieShuffleBot

A telegram bot based on [tgbotplug].

Find it here : [indieshufflebot]

### Functionalities

- Get latest songs from [indieshuffle]. 
- Get popular songs from [indieshuffle].
- Get Song of the day from [indieshuffle].
- More coming soon...
